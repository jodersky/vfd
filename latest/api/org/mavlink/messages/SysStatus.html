The general system state. If the system is following the MAVLink standard, the system state is mainl y defined by three orthogonal states/modes: The system mode, which is either LOCKED (motors shut dow n and locked), MANUAL (system under RC control), GUIDED (system with autonomous position control, po sition setpoint controlled manually) or AUTO (system guided by path/waypoint planner). The NAV_MODE defined the current flight state: LIFTOFF (often an open-loop maneuver), LANDING, WAYPOINTS or VECTO R. This represents the internal navigation state machine. The system status shows wether the system is currently active or not and if an emergency occured. During the CRITICAL and EMERGENCY states the MAV is still considered to be active, but should start emergency procedures autonomously. After a f ailure occured it should first move from active to critical to allow manual intervention and then mo ve to emergency after a certain timeout.
