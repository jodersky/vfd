case class HilOpticalFlow(timeUsec: Long, sensorId: Byte, integrationTimeUs: Int, integratedX: Float, integratedY: Float, integratedXgyro: Float, integratedYgyro: Float, integratedZgyro: Float, temperature: Short, quality: Byte, timeDeltaDistanceUs: Int, distance: Float) extends Message with Product with Serializable

Simulated optical flow from a flow sensor (e.g. PX4FLOW or optical mouse sensor)

temperature

Temperature * 100 in centi-degrees Celsius

quality

Optical flow quality / confidence. 0: no valid flow, 255: maximum quality

distance

Distance to the center of the flow field in meters. Positive value (including zero): distance known. Negative value: Unknown distance.
