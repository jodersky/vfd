case class RcChannelsRaw(timeBootMs: Int, port: Byte, chan1Raw: Short, chan2Raw: Short, chan3Raw: Short, chan4Raw: Short, chan5Raw: Short, chan6Raw: Short, chan7Raw: Short, chan8Raw: Short, rssi: Byte) extends Message with Product with Serializable

The RAW values of the RC channels received. The standard PPM modulation is as follows: 1000 microsec onds: 0%, 2000 microseconds: 100%. Individual receivers/transmitters might violate this specificatio n.

port

Servo output port (set of 8 outputs = 1 port). Most MAVs will just use one, but this allows for more than 8 servos.

rssi

Receive signal strength indicator, 0: 0%, 100: 100%, 255: invalid/unknown.
