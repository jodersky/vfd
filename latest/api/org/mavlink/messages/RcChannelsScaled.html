case class RcChannelsScaled(timeBootMs: Int, port: Byte, chan1Scaled: Short, chan2Scaled: Short, chan3Scaled: Short, chan4Scaled: Short, chan5Scaled: Short, chan6Scaled: Short, chan7Scaled: Short, chan8Scaled: Short, rssi: Byte) extends Message with Product with Serializable

The scaled values of the RC channels received. (-100%) -10000, (0%) 0, (100%) 10000. Channels that a re inactive should be set to UINT16_MAX.

port

Servo output port (set of 8 outputs = 1 port). Most MAVs will just use one, but this allows for more than 8 servos.

rssi

Receive signal strength indicator, 0: 0%, 100: 100%, 255: invalid/unknown.
